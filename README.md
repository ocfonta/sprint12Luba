### Как развернуть

Клонировать репозиторий `git clone`  <br/>
Установить npm `npm i`  <br/>
Запустить сборку:  <br/>
`npm run build` - сборка Production  <br/>
`npm run dev` - сборка Development, hot reload <br/>
