### Как развернуть

Клонировать репозиторий `git clone`  <br/>
Установить npm `npm i`  <br/>
Запустить сборку:  <br/>
`npm run start` - запуск на localhost 3000 без hot-reload <br/>
`npm run dev` - запуск на localhost 3000 c hot-reload <br/>
